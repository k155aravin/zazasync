# ZazaSync 🌿

> **Quebec Cannabis. Synchronized.**

ZazaSync is a live cannabis intelligence platform built around real-time Quebec market data. The platform continuously synchronizes publicly available SQDC product information and delivers it through a clean, modern, data-driven experience.

## 💡 What is ZazaSync?

ZazaSync combines cannabis culture aesthetics with a Bloomberg Terminal-style data platform. Built for Quebec cannabis consumers who want:

- **Real-time product availability** across all 107 SQDC locations
- **Price tracking** and drop alerts
- **Market analytics** — trending products, category breakdowns
- **Restock notifications** — never drive to an empty store again
- **Quebec availability map** — see which store has what, right now

---

## ✨ Features

### 🖥️ Desktop Platform
- Hero section with live sync stats and radar animation
- Full product discovery with search and category filters
- Interactive Quebec store availability map
- Live sync terminal — auto-updating activity log
- Market analytics dashboard — charts, trending, insights
- API section for developers
- Scanline and grid cyber aesthetic

### 📱 Mobile App Experience
- Native app feel — bottom tab navigation
- 5 screens: Home, Map, Trending, Alerts, Profile
- Tap-to-open product detail sheets
- Real-time search filtering
- Store availability map with live pins
- Alert system — restock, price drops, new products
- Watchlist and personalization
- Designed for one-thumb use
